Wordle Game & Solver 


 Description
This is a simple Wordle game in C.  
You can either play the game yourself or let the program try to solve the word automatically.  
The program uses a dictionary file the_secret_words.txt.

---

Requirements
- C Compiler (like gcc)  
- Dictionary file: the_secret_words.txt (5-letter words)

---

 How to Run
1. Compile the program:
```bash
gcc wordle.c -o wordle


Run:
wordle.exe   

How to play:
Select mode:
1. Play Wordle
2. Solver
